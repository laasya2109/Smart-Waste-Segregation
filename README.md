# EcoSort AI: Smart Waste Segregation

EcoSort AI is an intelligent waste classification and educational platform designed to make recycling smart, interactive, and gamified.

## 🚀 Key Features
* **AI Waste Classification**: Instantly identify waste items via webcam capture or image uploads.
* **Llama-3.2-Vision Refinement**: Multi-stage classification that uses advanced vision LLMs to resolve generic categories (e.g. mapping "plastic" to "biscuit wrapper").
* **Recycling Facility Locator**: Interactive map (Leaflet.js) directing users to nearby recycling sites.
* **Eco-Play Hub**: A drag-and-drop educational game testing waste-sorting knowledge.
* **EcoBot AI Chat**: A natural language chatbot assisting with sorting guidelines.

## 🛠️ Technologies Used
* **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (ES6)
* **Backend**: Flask (Python), SQLite3 Database
* **AI Models**: YOLOv8 Object Detection, Llama-3.2-Vision (Groq API)
* **Interactive Map**: Leaflet.js (OpenStreetMap)
